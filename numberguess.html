Write a program called NumberGuess to play the number guessing game. The program shall generate a
random number between 0 and 99. The player inputs his/her guess, and the program shall response with
"Try higher", "Try lower" or "You got it in n trials" accordingly. For example:
> java NumberGuess
Key in your guess:
50
Try higher
70
Try lower
65
Try lower
"
You got it in 4 trials!
Hints: Use java.util.Random class/ Math.random to produce a random number.

Code

import java.util.Scanner;
//To create a random class
import java.util.Random;
class NumberGuess
{
public static void main(String[] a)
{
//initialize the random class
Random n=new Random();
//generate random variable
int v=n.nextInt(100);
int c=0;
Scanner inp=new Scanner(System.in);
int x=inp.nextInt();
//check weather the random value is equals to guess
while(x!=v)
{
if(x>v)
{
System.out.println("Try lower");
c++;
x=inp.nextInt();
}
if(x < v)
{
System.out.println("Try greater");
c++;
x=inp.nextInt();
}
}
c++;
System.out.println("You got in "+c+" trails...");
}
}
