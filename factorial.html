To find the factorial of number.


Code

import java.util.Scanner;
class fact
{
public static void main(String[] a)
{
int n,fact,i;
fact=1;
Scanner inp=new Scanner(System.in);
n=inp.nextInt();
for(i=1;i<=n;i++)
{
fact*=i;
}
System.out.println(fact);
}
}

Output


5
120
