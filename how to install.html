How to install Java in your PC?


Step1:Click on the link below

Click here


Step2:Accept the license agreement and download the version according to your PC configuration.In my case it is windows 64 bit.




Step3:If you have account login to it or else signup to oracle.After login,download will start automtically




Step4:Run the program after downloading.Follow the instructions and install the software


Step5:After installation a folder named Java is created in program files.Click on the file and choose jdk 1.8.0_211 and then bin and then copy the path of folder








Step6:Copy the path of bin folder




Step7:Now,open control panel,click on System and security then System and then Advanced System settings







Step8:In advanced System settings choose environment variables



Step9:In environment variables under System variables double click on path and add new path and paste the copied java path in program files to it and click ok

Note:The path is "C:\Program Files\Java\jdk1.8.0_211\bin" for almost all PC's.you can copy this and paste




You can check whether java is successfully installed or not by opening commandprompt and type javac.If you get output as below Java is successfully installed



Hurray!!You have successfully installed java.Good Luck and happy coding
